说明

  • VUE_APP_API_BASE_URL 应该是: http://localhost:3000/api/v1
  • VUE_APP_SERVER_URL 应该是: http://localhost:3000
  • 如果值不正确，请停止前端服务，然后重新启动
  • 修改 .env 文件后必须重启服务才能生效

如何修复

  1. 停止前端服务（在命令行窗口按 Ctrl+C）
  2. 确认 frontend/.env.development 文件内容正确
  3. 重新启动前端服务（npm run serve）
  4. 清除浏览器缓存（Ctrl+Shift+Delete）
  5. 刷新此页面（Ctrl+F5）
